# 第2部：配色

色は小紅書のビジュアル的な魅力の核心です。よく選ばれたカラーパレットは、あなたのスタイルと雰囲気を瞬時に伝え、ユーザーのスクロールを止めてカバーに注目させます。このパートでは、色彩理論の基礎からトレンドパレット、配色ルール、独自のカラースキームを構築するための実践的なツールまでを解説します。

## チャプター概要

### 2.1 色彩理論

色相環、HSBモデル、色温度、色彩心理学――これらの基礎を理解することが、優れたパレットへの第一歩です。ヘックスコードを暗記する必要はありませんが、*なぜ*特定の色の組み合わせがうまくいくのかを理解しましょう。

**キーコンセプト：** 色相環、原色と補色、HSBモデル、色温度、色彩心理学

---

### 2.2 スタイル別パレット

モランディトーンからクリーム色のパステル、高彩度コントラストからダーク＆ゴールドまで――スタイル別に整理されたパレット集。各パレットにはヘックスコードと推奨用途を掲載しています。

**キーコンセプト：** モランディパレット、高彩度コントラスト、クリームトーン、ダーク＆ゴールド

---

### 2.3 配色ルール

60-30-10の法則、同系色グラデーション、補色コントラスト――調和と奥行きを両立するパレットを作るための原則をマスターしましょう。よくある落とし穴についても解説します。

**キーコンセプト：** 60-30-10の法則、同系色配色、補色コントラスト、よくある落とし穴

---

### 2.4 ツールと実践

モバイルアプリからオンラインプラットフォームまで、優れたカラーツールのまとめ。学んだすべてを毎回の投稿に活かすためのステップバイステップワークフローも紹介します。

**キーコンセプト：** 醒図（Xingtu）、Canva、Coolors、Adobe Color、黄油相機

---

## 学習パス

チャプターを順番に、理論から実践へと進むことをおすすめします：

```
2.1 色彩理論（基礎）
    ↓
2.2 スタイル別パレット（インスピレーションライブラリ）
    ↓
2.3 配色ルール（方法論）
    ↓
2.4 ツールと実践（実行）
```

!!! tip "ポイント"
    時間がない方は **2.2 スタイル別パレット** から始めて、自分のスタイルに合うパレットを見つけてから、あとで理論に戻りましょう。

# Part 6: Style by Category

Different content categories have different visual languages. What looks "good" for a food post is completely different from a coding post — color schemes, composition, layout, and title formulas all differ.

This Part breaks down the optimal visual style for four major Xiaohongshu categories, each with full phone mockup demos.

---

## Chapter Overview

### 6.1 Tech & Coding

Dark backgrounds + bright titles + tech tags give code posts a premium feel. Full breakdown of GitHub Dark, Dracula, and Solarized color schemes.

**Keywords:** Dark themes · Code aesthetics · Open source tags

---

### 6.2 Food & Cafe

Warm tones reign supreme. 45-degree angles, creamy palettes, and visual formulas that make viewers hungry.

**Keywords:** Warm tones · Food close-ups · Atmosphere

---

### 6.3 Travel

Intentional composition, unified color grading, and mood. City exploration, natural landscapes, and hidden gems each have different approaches.

**Keywords:** Composition first · Unified tones · Destination mood

---

### 6.4 Fashion & Beauty

Clean backgrounds, full-body framing, tones that match the outfit. OOTD, beauty tutorials, and outfit collections each have templates.

**Keywords:** Clean backgrounds · Portrait framing · Product aesthetics

---

## How to Use

!!! tip "Start with your category"
    No need to read everything in order. Start with the category you focus on, then cross-reference others for inspiration.

Every chapter leads with **phone mockup demos** — see the result first, learn the method second.

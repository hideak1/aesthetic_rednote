# 2.4 ツールと実践

理論とルールが揃いました。最後のステップは、実際のコンテンツ制作に活かすことです。このチャプターでは実用的なカラーツールを紹介し、完全なワークフローを解説します。

## おすすめツール

### 醒図（Xingtu）

醒図は小紅書クリエイターの間で最も人気のある写真編集アプリの一つで、フィルターと色調整機能が充実しています。

!!! tip "ポイント"
    - HSLパネルで各色相の彩度と明度を個別に調整でき、画像のトーンを精密にコントロールできます
    - 内蔵フィルターでワンタップで画像全体の色調を統一
    - 「テンプレート」機能で既成のカラースキームが使えます――初心者に最適

**おすすめ用途：** 写真のカラーグレーディング、フィルターによるトーン統一、クイック編集

---

### Canva

Canvaはオンラインデザインプラットフォームで、最大の強みは**テンプレートに配色が組み込まれている**ことです。

- 「小紅書」で検索すると、プラットフォームに最適化されたサイズのテンプレートが見つかります
- すべてのテンプレートにプリセットパレットあり。カラーブロックをクリックで配色全体を確認
- カスタムカラー置換に対応。気に入った色のテンプレートを見つけて、自分のコンテンツに差し替え

!!! info "コンセプト"
    好きな画像をアップロードすると、Canvaが自動的にその画像の支配色をパレットとして抽出してくれます。配色のインスピレーションを得る素晴らしい方法です。

**おすすめ用途：** カバーデザイン、テンプレートからのパレット参照、クイックレイアウト

---

### Coolors

[Coolors](https://coolors.co) はプロフェッショナルなオンラインパレットジェネレーターです。

- スペースバーを押すと5色のパレットをランダム生成
- 気に入った色をロックして、残りだけ再生成
- アップロード画像からのパレット抽出にも対応
- 色覚バリアフリーチェッカー付き

**おすすめ用途：** パレットのインスピレーション探し、新しい組み合わせの発見、プロフェッショナルなカラーワーク

---

### Adobe Color

[Adobe Color](https://color.adobe.com) はAdobeが提供する強力なカラーツールです。

- インタラクティブな色相環で「補色」「類似色」「トライアド」などのルールを選択して自動的にパレットを生成
- 「探索」ページでトレンドのカラーテーマを閲覧
- 画像からのパレット抽出に対応
- アクセシビリティ基準に基づく色のコントラストチェック

!!! example "実例"
    メインカラーは決まったけれど何と組み合わせるか迷ったとき、Adobe Colorにその色を入力して「補色」や「類似色」を選びましょう。調和のとれたパレットが自動生成されます。

**おすすめ用途：** 色相環ベースの体系的なパレット作成、コントラストチェック、プロフェッショナルなデザイン

---

### 黄油相機（Butter Camera）

黄油相機の強みは**タイポグラフィとテキストオーバーレイ**。カラー機能も充実しています。

- 豊富なフォントテンプレートライブラリ、それぞれに推奨カラーペアリング付き
- ステッカーとフィルターで全体の画像トーンを統一
- カスタムテキストカラーにヘックスコードの直接入力対応

**おすすめ用途：** テキストレイアウト、カバーへのタイトルオーバーレイ、テキストカラーの一貫性

---

## ステップバイステップワークフロー

ツールが揃ったところで、配色の完全なワークフローを紹介します。カバーデザインのたびにこの5つのステップに従いましょう：

### ステップ1：スタイルを決める

自分に問いかけましょう：この投稿でどんな雰囲気を伝えたい？

| 目標の雰囲気 | おすすめパレット |
|-------------|-------------------|
| 洗練された、穏やかな | モランディトーン |
| 鮮やか、目を引く | 高彩度コントラスト |
| 柔らかい、癒される | クリームトーン |
| プロフェッショナル、高級感 | ダーク＆ゴールド |

### ステップ2：メインカラーを選ぶ

対応するパレットから、メインカラー（60%）を1色選びましょう。これが背景やベースカラーになります。

### ステップ3：サブカラーを選ぶ

同じパレットから、サブカラー（30%）を選びましょう。カラーブロック、装飾、小見出しなどの補助要素に使います。

### ステップ4：アクセントを設定する

メインカラーとコントラストのある色をアクセント（10%）として選びます。タイトル、ボタン、重要なハイライトに使用。

### ステップ5：フィルターで統一する

最後に、醒図や黄油相機で統一感のあるフィルターをかけ、すべての色を同じトーンファミリーに収めましょう。

!!! warning "注意"
    多くのクリエイターがステップ1〜4は完璧にこなすのに、ステップ5を飛ばしてしまいます。統一フィルターなしでは、完璧に選んだヘックスコードでも、写真とテキストのトーン感が合わず違和感が残ることがあります。

## まとめ

| ツール | 主な強み | 価格 |
|------|--------------|-------|
| 醒図 | 写真のカラーグレーディング、フィルター | 無料 / プレミアム |
| Canva | テンプレートパレット、デザインレイアウト | 無料 / Pro |
| Coolors | ランダムパレット生成 | 無料 / Pro |
| Adobe Color | 色相環ベースのパレット、コントラストチェック | 無料 |
| 黄油相機 | タイポグラフィ、テキストカラーのスタイリング | 無料 / プレミアム |

!!! tip "ポイント"
    ツールはあくまで補助。大切なのはセンスです。小紅書で高エンゲージメントの投稿を研究しましょう。どんな色を、どんな比率で、どんなスタイルで使っているか。観察すればするほど、配色のセンスは磨かれます。

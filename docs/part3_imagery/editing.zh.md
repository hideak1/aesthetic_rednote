# 3.2 修图与滤镜 — 统一风格，提升质感

选好图片只是第一步。通过后期调色和修图，你可以让图片的质感再上几个台阶。但修图是一把双刃剑——用好了锦上添花，用过了适得其反。

---

## 调色原则：统一，是最重要的事

修图最核心的原则只有一个：**统一**。

一篇笔记里的所有图片，色调必须保持一致。图1暖黄、图2冷蓝、图3高饱和、图4黑白——这种混搭会让整篇笔记看起来杂乱无章，严重影响阅读体验。

!!! tip "如何做到统一"
    - 同一篇笔记使用同一个滤镜或同一组调色参数
    - 如果素材拍摄条件差异大，先统一白平衡和曝光，再加滤镜
    - 保存常用的调色预设，形成自己的固定风格

统一的色调不仅让单篇笔记更好看，还能让你的主页形成统一的视觉风格。当用户点进你的主页，看到整齐统一的色调，会大大提升关注的意愿。

## 常用滤镜风格

小红书上最受欢迎的几种滤镜风格，各有各的适用场景：

### 日系风格

**特点：** 高亮度、低饱和度、偏暖色调、整体柔和

适合品类：日常生活、美食、文艺穿搭、咖啡探店

!!! example "调色方向"
    曝光略微提高，对比度降低，饱和度降低，色温偏暖，高光偏黄，阴影提亮。整体追求一种"柔柔的、暖暖的"感觉。

### 韩系风格

**特点：** 冷色调、清透感、低饱和、干净利落

适合品类：护肤、简约穿搭、极简家居、氛围感内容

!!! example "调色方向"
    色温偏冷，饱和度降低，对比度适中，高光偏蓝白，肤色保持自然。追求"冷冷的高级感"。

### 复古风格

**特点：** 暖黄色调、低对比度、略带颗粒感

适合品类：旅行、街拍、胶片感日常、vintage穿搭

!!! example "调色方向"
    色温偏暖，阴影偏黄绿，高光偏橙，对比度降低，加少量颗粒。模拟胶片相机的质感。

### 高级灰风格

**特点：** 低饱和度、高质感、色彩克制、整体偏灰调

适合品类：高端产品、极简风格、建筑摄影、质感穿搭

!!! example "调色方向"
    大幅降低饱和度，对比度适中偏高，色调整体偏灰，保留少量主色调。追求"less is more"的克制美感。

## 修图工具推荐

不同的工具适合不同的需求和水平：

| 工具 | 优势 | 适合人群 | 平台 |
|------|------|----------|------|
| 醒图 | 功能全面，模板丰富，中文界面友好 | 入门到进阶 | iOS / Android |
| VSCO | 滤镜质量极高，色彩审美在线 | 追求质感的用户 | iOS / Android |
| Lightroom | 专业级调色，精细参数控制 | 进阶到专业 | iOS / Android / 桌面 |
| Snapseed | Google出品，局部调整功能强 | 需要精修的用户 | iOS / Android |

!!! tip "新手建议"
    如果你刚开始学修图，推荐从醒图入手。它的滤镜模板可以一键套用，调色参数也可以手动微调。等熟悉了调色逻辑之后，再尝试 VSCO 或 Lightroom。

## 修图避雷：这些坑不要踩

修图最怕用力过猛。以下是最常见的翻车情况：

### 磨皮过度

!!! warning "假脸警告"
    磨皮到鼻子都没有了、皮肤像塑料一样光滑——这种修图不仅不好看，还会让人产生不信任感。保留适度的皮肤纹理，才是真实又好看的修图。

### 滤镜不统一

同一篇笔记里图片风格不统一是大忌。就算每张图单独看都不错，放在一起风格乱跳也会让观感大打折扣。

### 亮度过高或过低

过曝让画面发白、细节丢失；过暗让人看不清内容。修图时在手机上看着合适，建议再降低屏幕亮度检查一次——很多人的手机亮度长期调得很高，导致修出来的图在别人手机上偏暗。

### 过度锐化

适度锐化可以让图片更清晰，但过度锐化会让边缘出现不自然的白边和噪点，画面看起来很"硬"、很"假"。

!!! tip "修图的最高境界"
    让人觉得"这张图真好看"，但说不出来哪里修过——这就是最好的修图。好的后期是隐形的，它提升质感但不留痕迹。

## 本章小结

- **调色第一原则**：统一，统一，还是统一
- **找到适合自己的滤镜风格**：日系、韩系、复古、高级灰各有所长
- **工具不必贪多**：精通一个比泛泛用十个强
- **修图克制**：过度修图比不修图更可怕

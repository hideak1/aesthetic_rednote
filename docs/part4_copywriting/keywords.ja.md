# 4.3 キーワードのエレガントな埋め込み

多くのクリエイターがレコメンドフィードのトラフィックだけに注目し、もう一つの巨大なチャネル――検索を見落としています。小紅書の検索エンジンは急成長中で、ユーザーは「秋コーデ」「プチプラスキンケア」「狭い部屋のリノベ」といったキーワードを積極的に検索しています。あなたの投稿にそれらのワードが含まれていれば、検索結果に表示される可能性があります。しかも検索トラフィックは、閲覧トラフィックよりもターゲットが明確でコンバージョン率も高い傾向にあります。

鍵となる問いは、こうです：可読性を犠牲にせずに、アルゴリズムに認識されるよう、キーワードをどう自然にテキストに織り込むか？

---

## なぜキーワードが重要なのか

!!! info "コンセプト"
    - **検索での発見：** ユーザーがキーワードを検索したとき、あなたの投稿が結果に表示される可能性がある
    - **レコメンドの精度向上：** アルゴリズムがキーワードでコンテンツを理解し、適切なオーディエンスにプッシュする
    - **ロングテールトラフィック：** 何ヶ月も前に公開した投稿でも、検索を通じて安定した露出を得続けることができる

小紅書では、検索トラフィックとレコメンドトラフィックがそれぞれ総インプレッションの約半分を占めます。検索最適化を無視するということは、潜在的な読者の半分を放棄しているのと同じです。

---

## キーワードの配置場所

位置によって重みが異なります。最も効果の高い場所にコアキーワードを優先的に配置しましょう。

!!! info "コンセプト"
    **タイトル（最も重み大）> 最初の段落 > 本文の残り > ハッシュタグ**

### タイトルにキーワードを入れる

タイトルは検索ランキングにおいて最も重要な位置です。コアキーワードは必ずここに含めましょう。

!!! example "実例"
    - 「小紅書 レイアウト」で上位表示したい → タイトル：「必読｜小紅書レイアウト初心者ガイド」
    - 「秋コーデ」で上位表示したい → タイトル：「通勤にぴったりの秋コーデ見つけた」

### 最初の段落

冒頭の段落は2番目に高い重みを持ちます。自然な形で最初にキーワードを導入しましょう。アルゴリズムがそれを拾い上げ、読者もすぐにトピックを理解できます。

!!! example "実例"
    「今日は超実用的な**小紅書のレイアウトテクニック**を共有します。特にこれから投稿を始める初心者の方にぴったりの内容です。」

### 本文の残り

本文全体でキーワードを繰り返すことで、トピックの関連性が強化されますが、必ず自然に読めるようにしましょう。

### ハッシュタグ

ハッシュタグはキーワードを補完し、アルゴリズムがコンテンツをさらに細かく分類するのに役立ちます。

---

## 自然な埋め込み：織り込む、詰め込まない

キーワードの最大のミスは「詰め込み」――広告バナーのようにキーワードを繰り返すことです。読み心地が台無しになるだけでなく、低品質コンテンツとしてフラグが立つ可能性もあります。

!!! warning "注意"
    - **詰め込み：** 「コーデ コーデ コーデ、今日はコーデを共有、コーデのおすすめ、コーデまとめ」――スパムのように読まれ、プラットフォームからペナルティを受ける
    - **自然な埋め込み：** 「この秋コーデは通勤にぴったり。シンプルで着痩せして、体型を選ばない」――「秋コーデ」「通勤」が文中に自然に存在する

!!! tip "ポイント"
    - キーワードを文の一部として扱い、独立した要素にしない
    - 同じトピックに類義語やバリエーションを使う：「コーデ」「ルック」「スタイリング」「着こなし」
    - コアキーワードの出現は1投稿あたり3〜5回で十分。毎文に入れる必要はない

!!! example "実例"
    「秋到来！ 私の定番**秋の通勤コーデ**を紹介します。この**着こなし**の核心はレイヤード。ブレザーにニットを合わせて、ワイドパンツでまとめる。全体の**スタイリング**はすっきりして、オフィスからディナーまでそのまま行けます。」

    「秋の通勤コーデ」「着こなし」「スタイリング」が異なる文に分散。テキストはスムーズに流れ、アルゴリズムもトピックを識別できます。

## ロングテールキーワード戦略：具体的なほど有利

「ロングテールキーワード」とは、より具体的でニッチな検索ワードです。大きなキーワードより検索量は少ないですが、競合がはるかに少ないため、上位表示が容易です。

!!! info "コンセプト"
    - **大きなキーワード：** 「コーデ」――検索量は膨大だが競争が激しく、新しい投稿が上位に入るのはほぼ不可能
    - **ロングテールキーワード：** 「低身長 秋冬コーデ」――適度な検索量、低競争、検索者の意図が明確

!!! example "実例"
    | 大きなキーワード | ロングテールキーワード |
    |---|---|
    | コーデ | 低身長 秋冬コーデ、通勤コーデアイデア、ぽっちゃり着痩せコーデ |
    | スキンケア | プチプラ脂性肌スキンケア、敏感肌 冬のルーティン、学生 コスパスキンケア |
    | インテリア | 狭い部屋のリノベ、100万円以下のリフォーム、賃貸でもできる模様替え |

!!! tip "ポイント"
    - 小紅書の検索バーを開き、大きなキーワードを入力して自動候補をメモする
    - 同ジャンルのバズ投稿のタイトルを研究する。どんな具体的フレーズが使われているか？
    - ターゲット読者の立場に立つ。「自分なら何と入力する？」より具体的であるほど良い

---

## ハッシュタグ戦略

ハッシュタグ（トピックタグ）はキーワードエコシステムの重要な一部です。適切な組み合わせでコンテンツの露出を最大化しましょう。

!!! tip "ポイント"
    **精密タグ3〜5個 + トレンドタグ1〜2個 + カテゴリタグ1個**

!!! example "実例"
    「低身長 秋冬コーデ」の投稿の場合：

    - **精密タグ：** #低身長コーデ #秋冬コーデ #通勤コーデ #スタイルアップ
    - **トレンドタグ：** #OOTD #今日のコーデ
    - **カテゴリタグ：** #ファッション

!!! warning "注意"
    - メガ人気タグだけに頼らない（#コーデ のような）。投稿が埋もれてしまう
    - トラフィック目当てで無関係なタグを付けない。アルゴリズムはミスマッチを検出する
    - タグ数は合計5〜8個に。多すぎると散漫に見える
    - 複数のタグで同じ単語を繰り返さない

---

## チャプターまとめ

キーワード最適化はテクニカルな作業ではなく、共感力の作業です。読者の立場に立って「このトピックを探すなら何と入力する？」と考え、その言葉をタイトルと本文に自然に織り込みましょう。

!!! tip "ポイント"
    1. よく書くジャンルを1つ選ぶ（例：コーデ、スキンケア、グルメ）
    2. 小紅書の検索バーでコアキーワードを入力し、自動候補を10個メモする
    3. ロングテールキーワードを2〜3個選び、それらを自然に含むタイトル＋本文3段落を書いてみる

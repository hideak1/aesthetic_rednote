# Part 3: 配图 — 让人忍不住点进来的视觉密码

**好内容配上好图片，才能真正被看见。这一部分带你掌握选图、修图、编排和封面设计的核心技巧。**

---

## 概述

在小红书，图片就是第一生产力。一篇笔记能不能被点开，80%取决于封面图；能不能被看完，取决于每一张图的质量和编排逻辑。配图不是随手拍、随手选，而是一套完整的视觉策略。

这一部分将从选图标准、修图调色、多图编排到封面类型，系统性地帮你建立配图思维。

## 你将学到

- 什么样的图片值得被选中
- 如何通过修图和滤镜统一视觉风格
- 多图笔记的编排逻辑与节奏感
- 不同品类适合什么类型的封面图

## 学习建议

建议按顺序学习：先掌握选图的底层标准，再学调色统一风格，然后理解多图编排的节奏，最后根据自己的品类选择合适的封面类型。

每学完一个章节，试着用自己的素材实操一次，效果会比单纯阅读好得多。

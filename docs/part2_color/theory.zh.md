# 2.1 色彩基础

在小红书上，颜色往往是用户对一篇笔记的「第一印象」。在你开始挑选色号之前，先花几分钟了解色彩的基本原理——这比死记任何一组色值都有用得多。

## 色轮基础

色轮（Color Wheel）是理解颜色关系的基础工具。

!!! info "三原色"
    色轮的核心是三原色：**红（Red）**、**黄（Yellow）**、**蓝（Blue）**。所有其他颜色都由这三种颜色混合而来。

**间色**（Secondary Colors）是两种原色等量混合的产物：

| 混合方式 | 间色 |
|---------|------|
| 红 + 黄 | 橙色 |
| 黄 + 蓝 | 绿色 |
| 红 + 蓝 | 紫色 |

**补色**（Complementary Colors）是色轮上正对面的两种颜色，比如红与绿、蓝与橙、黄与紫。补色放在一起会产生强烈的视觉对比——用好了是「撞色高级感」，用不好就是「土味翻车」。

!!! tip "记住这个规律"
    色轮上距离越近的颜色越和谐，距离越远的颜色对比越强。配色时先想想你要的是「和谐感」还是「冲击力」。

## HSB 模型

比起记住具体的色号（比如 #8EA4B5），理解 HSB 三个维度对配色帮助更大。

**H — 色相（Hue）**

色相就是我们平时说的「什么颜色」——红、橙、黄、绿、蓝、紫。它对应色轮上的角度，从 0° 到 360°。

**S — 饱和度（Saturation）**

饱和度决定颜色的「鲜艳程度」。饱和度越高颜色越鲜艳浓烈，越低则越灰、越柔和。

| 饱和度 | 效果 | 适合风格 |
|--------|------|---------|
| 高（80-100%） | 鲜艳、活泼、抓眼球 | 美妆、美食、潮流 |
| 中（40-70%） | 舒适、自然 | 穿搭、旅行、家居 |
| 低（10-30%） | 高级、安静、灰调 | 莫兰迪、侘寂风 |

**B — 明度（Brightness）**

明度控制颜色的「亮暗」。高明度偏白偏亮，低明度偏黑偏暗。

!!! example "实际应用"
    同样是蓝色，高饱和高明度 = 克莱因蓝（活力十足），低饱和中明度 = 雾霾蓝（温柔安静）。理解了 HSB，你就能从一个颜色「推导」出一整组色板。

## 色温

颜色有「温度」之分，这直接影响画面给人的感受。

**暖色调：红、橙、黄**

- 给人温暖、热情、活力、亲切的感觉
- 适合美食、穿搭、家居、生活类内容
- 能拉近与观众的心理距离

**冷色调：蓝、绿、紫**

- 给人冷静、理性、清新、高级的感觉
- 适合科技、商务、旅行、极简风内容
- 营造空间感和距离感

!!! info "小红书趋势"
    近两年小红书上「冷暖混搭」非常流行——比如暖调画面 + 冷调文字，或冷调滤镜 + 暖色点缀。这种混搭能让画面既有温度又不失高级感。

## 色彩心理

每种颜色都会无意识地唤起特定的情绪和联想。了解这些心理效应，能帮你用颜色「说话」。

| 颜色 | 心理联想 | 适合内容 |
|------|---------|---------|
| **红色** | 热情、活力、紧迫感、喜庆 | 促销、美妆、节日 |
| **蓝色** | 冷静、信任、专业、广阔 | 科技、商务、旅行 |
| **绿色** | 自然、健康、清新、放松 | 健身、美食、户外 |
| **粉色** | 温柔、甜美、浪漫、少女感 | 美妆、穿搭、恋爱 |
| **黄色** | 快乐、活泼、引人注目 | 美食、亲子、学习 |
| **黑色** | 高级、神秘、力量感 | 数码、奢侈品、男性向 |
| **白色** | 干净、简约、留白感 | 极简风、日系、医美 |

!!! warning "注意"
    色彩心理不是绝对的——同一种颜色在不同饱和度、明度、搭配下传达的情绪完全不同。比如正红色是热情大气，而暗红色可能给人沉闷感。要结合具体场景灵活运用。

## 小结

理解色轮关系、掌握 HSB 三维度、感知色温差异、了解色彩心理——有了这些基本功，你就不再是「凭感觉挑颜色」，而是「有理有据地选配色」。下一章我们会直接给你按风格整理好的色板，拿来即用。

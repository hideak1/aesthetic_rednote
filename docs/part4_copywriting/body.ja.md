## 段落のリズム：短文と長文の交互

良い本文のリズムは呼吸のようなもの。短い文が息を吐く瞬間、パンチがあってエネルギッシュ。長い文が息を吸う瞬間、展開と説明を担う。この交互が読みやすさを生みます。

!!! example "実例"
    **リズムなし：**
    「今日はカバーデザインのコツを共有したいんですが実はカバーデザインって皆が思っているほど難しくなくていくつかのコアな原則を知るだけで素敵なカバーが作れるようになるんです。」

    **リズムあり：**
    「カバーデザインって難しい？ そうでもないです。いくつかの基本原則を押さえれば、初心者でも洗練されたカバーが作れます。今日は、何ヶ月もかけて検証したテクニックを紹介します。シンプルで、実践的で、すぐに使えるものばかり。」

!!! tip "ポイント"
    - 一文はおよそ20文字以内が最も読みやすい
    - 重要なポイントは短文で、説明は長文で
    - 2〜3文ごとに改行し、視覚的な疲れを防ぐ

## セクション区切り：テキストのビジュアルパンクチュエーション

小紅書の投稿は従来の記事より「ビジュアルノート」に近いです。適切な区切りスタイルで情報の階層を一目でわかるようにしましょう。

### パターンA：絵文字区切り

各段落の頭に絵文字をビジュアルアンカーとして使用。カラフルで親しみやすく、ライフスタイルや商品紹介の投稿に最適。

!!! example "実例"
    ✨ ステップ1：メインカラーを決める

    🎨 ステップ2：サブカラーを1〜2色追加

    💡 ステップ3：ニュートラルカラーを背景に

!!! info "コンセプト"
    商品まとめ、チュートリアル、日常シェアに最適。絵文字のスタイルは統一しましょう。花、星、動物をランダムに混ぜないこと。

### パターンB：空行区切り

最もクリーンなアプローチ。情報ブロックを空行で分離するだけ。

!!! example "実例"
    今日はカバーレイアウトのコツを3つ共有します。

    まず、テキストを詰め込まない。カバーにはキーワード1〜2行で十分。

    次に、余白を大切に。要素をギリギリまで詰めない。

    最後に、揃える。テキストも画像も一貫した方向で揃えること。

### パターンC：番号リスト

ハウツーコンテンツやステップバイステップガイドに最適。論理構造が明確で、読者が情報を素早く見つけられます。

!!! example "実例"
    1. テンプレートを選ぶ
    2. 自分の画像とテキストに差し替える
    3. カラーを自分のスタイルに合わせて調整
    4. 書き出して公開

!!! tip "ポイント"
    - ライフスタイル・商品投稿 → 絵文字区切り
    - 意見・エモーショナル系投稿 → 空行区切り
    - チュートリアル・ハウツー投稿 → 番号リスト
    - 1つの投稿内では1つのスタイルに統一

## 情報密度：1段落1ポイント

初心者に最も多いミス――1つの段落に3〜4つのアイデアを詰め込むこと。読むのが疲れるし、何も記憶に残りません。

!!! warning "注意"
    **詰め込み段落：**
    「カバーは色に注意して、テキストも多すぎないようにして、余白も残して、あとフォントも大事で源ノ角ゴシックか站酷フォントがおすすめで、レイアウトは中央揃えがいいです。」

    **1段落1ポイント：**
    「カバーの色：2〜3色で十分。多すぎると散漫に見えます。」
    （改段落）
    「テキストは最小限に：カバーにはキー情報を1〜2行だけ。それ以上だと何も目立ちません。」

!!! info "コンセプト"
    各段落を付箋だと思いましょう。1枚の付箋に1つのアイデア。「あと」「それから」「ちなみに」が続いたら、段落を分割するタイミングです。

## 視覚的な余白：洗練された見た目の秘密

!!! warning "注意"
    - **テキストの壁：** 長い段落、空行なし、絵文字の乱用、情報が密集 → 読者はすぐに離脱
    - **余白のあるレイアウト：** 短い段落、セクション間に空行、明確な情報ポイント、視覚的に快適 → 読者は最後まで読む

!!! tip "ポイント"
    - 1段落は3〜4行以内
    - 段落間に1行の空白
    - 重要なポイントは独立した段落に
    - 絵文字は連続3つまで

## 書き出し：最初の2行が勝負

タップした後、読者の目は本文の最初の2行に向かいます。この2行で引き留められなければ、離脱されます。

!!! tip "ポイント"
    - **共感を生む：** 「毎回カバーで悩んでいませんか？ 私もそうでした…」
    - **好奇心を刺激：** 「この方法を使い始めてから、保存数が3倍になりました。」
    - **価値を先に提示：** 「今日は6ヶ月間使い続けたレイアウトの5つのコツを共有します。すべて実証済み。」

!!! warning "注意"
    - 「みなさんこんにちは、今日は共有したいことが…」――平坦すぎ、情報がゼロ
    - 「よく聞かれるので…」――大物クリエイターでなければ、読者は気にしない

---

## 締めくくり：アクションを促す

締めくくりは「最後の一文」ではなく、読者にアクションを起こさせる最後のチャンスです。

!!! tip "ポイント"
    - **質問する：** 「カバー作りにどのアプリ使ってますか？ コメントで教えて！」
    - **投票を作る：** 「AとBどっちが好き？」
    - **保存を促す：** 「参考になったら、次のカバー作りのために保存しておいてね！」
    - **次回を予告：** 「次は配色のコツを紹介します。お見逃しなく！」

---

## チャプターまとめ

本文のレイアウトは「書いて投稿するだけ」ではなく、カバーデザインと同等の丁寧さが必要です。リズム、区切り、密度、余白、書き出し、締めくくり――すべての要素が、読者が最後まで読むかスクロールして去るかを左右します。

!!! tip "ポイント"
    スラスラ読めるバズ投稿を見つけたら、段落ごとに分析してみましょう。どんな区切りスタイルを使っている？ 1段落は何行？ 書き出しは？ 締めくくりでどうアクションを促している？

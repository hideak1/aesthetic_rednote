# 3.4 不同类型封面图 — 选对类型，点击率翻倍

不同内容品类适合完全不同的封面类型。选对类型比单纯提高画质更能提升点击率。

---

## 五大封面类型一览

| 类型 | 特征 | 适合品类 | 点击力 |
|------|------|----------|--------|
| 人物出镜 | 面部清晰、表情有感染力 | 穿搭、美妆、vlog | 高 |
| 产品平铺 | 物品整齐排列、俯拍 | 好物分享、开箱 | 中高 |
| 对比图 | 左右/上下 before-after | 教程、测评、改造 | 高 |
| 文字海报 | 大字报风格、信息前置 | 干货、攻略、清单 | 高 |
| 场景氛围 | 生活场景、有代入感 | 家居、旅行、美食 | 中 |

## 对比图：天然自带吸引力

<div class="cover-mockup" style="display: grid; grid-template-columns: 1fr 1fr; gap: 0;">
  <div style="background: #ddd; display: flex; align-items: center; justify-content: center; font-size: 0.75rem; color: #888; border-right: 1px dashed var(--border-color);">Before</div>
  <div style="background: #FCE4EC; display: flex; align-items: center; justify-content: center; font-size: 0.75rem; color: #880E4F;">After</div>
</div>

人对"变化"有本能好奇心。一张图同时展示"之前"和"之后"，视觉冲击力极强。关键是对比要明显、拍摄条件一致、分割清晰。

!!! tip "适用场景"
    化妆教程、护肤测评、房间改造、穿搭对比、收纳整理。

!!! warning "诚信底线"
    不要通过不同光线、角度故意夸大效果，用户一旦觉得作假，反噬严重。

## 文字海报：信息前置的利器

!!! tip "适用场景"
    干货教程、攻略合集、知识分享、避雷清单。对"收藏型"内容特别有效。

## 如何选择适合自己的封面类型？

考虑三个因素：**内容品类**（参考上面的适用场景）、**竞争对手**（看同品类热门笔记的封面类型）、**个人优势**（擅长出镜就多用人物出镜，擅长设计就多用文字海报）。

!!! tip "终极建议"
    不要只用一种封面类型，根据内容灵活切换。但同品类内容尽量保持风格一致。

---

## 本章小结

- **人物出镜和对比图**点击力最高，对素材质量要求也最高
- **文字海报**是干货类内容的利器
- **产品平铺和场景氛围**更考验审美和构图能力
- 选封面类型时，结合品类、竞品和个人优势综合判断

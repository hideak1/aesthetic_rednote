# 第一部分：封面设计

封面是你在小红书上的"门面"——用户刷到你的笔记，决定是否点进来，往往只需要 0.3 秒。一张好的封面，不需要多花哨，但一定要**信息清晰、视觉舒服、有点击欲望**。这一部分，我们从构图、字体排版、信息布局三个维度，帮你建立系统的封面设计思维。

## 章节

### 1.1 构图法则

掌握最实用的构图技巧，让你的封面在信息流中脱颖而出。

**你将学到：**

- 三分法、居中对称、引导线等经典构图的使用场景
- 留白的重要性——为什么"少即是多"
- 如何用对比制造视觉层次，让画面更有冲击力
- 常见构图踩坑指南

---

### 1.2 字体与排版

学会选字体、定字号、排版面，让文字成为封面的加分项而不是减分项。

**你将学到：**

- 衬线与无衬线字体的区别，以及各自适用场景
- 标题、副标题、正文的字号比例建议
- 文字与图片如何和谐搭配
- 排版中最常犯的错误及避免方法

---

### 1.3 信息层级与布局

理解视觉动线和信息密度，让封面既好看又好懂。

**你将学到：**

- 主标题、副标题、正文、细节的层级关系
- Z 型阅读与 F 型阅读的适用场景
- 不同类型帖子（教程、种草、攻略）的布局策略
- 封面信息密度的把控原则

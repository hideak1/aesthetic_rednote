# 1.3 レイアウトと情報階層

前の2つのチャプターでは構図とタイポグラフィを扱いました。このチャプターはそれらを結びつけます――1枚のカバーに載るすべての情報をどう整理・配置するか。核心はシンプルです：**ユーザーがカバーを一瞬見ただけで、内容がわかるか？**

## 情報の階層

カバー上のすべての要素が同じ重要度ではありません。明確な優先順位が必要であり、それを確立するためのツールはシンプルです：**サイズ、ウェイト、色**。

### 4層構造

| レイヤー | 内容 | 視覚的な処理 | 例 |
|-------|---------|-----------------|---------|
| 第1層：見出し | カバーの核心的なフック | 最大・最太・最も目立つ | 「3ステップ朝食ハック」 |
| 第2層：小見出し | 補足的なコンテキスト | 中サイズ、やや控えめな色 | 「忙しい朝に・合計10分」 |
| 第3層：本文 | 重要な詳細 | 小さめのフォント、控えめに | 材料リスト、ステップ番号 |
| 第4層：補足情報 | 装飾的・補助的な情報 | 最小・最も薄く | ロゴ、日付、ウォーターマーク |

!!! tip "ポイント"
    カバーが完成したら、スマホを腕の長さほど離して見てみましょう。見出しが最初に目に入り、次に小見出しが見えるなら、階層はうまく機能しています。すべてがひとかたまりのテキストに見えるなら、調整が必要です。

!!! warning "注意"
    すべてを大きく、太く、鮮やかにすると、何も目立ちません――階層がないのと同じです。見出しを際立たせるために、補足情報は控えめにする勇気を持ちましょう。

## 視覚的な流れ

視覚的な流れとは、画像上で視線がたどる経路のことです。このパターンを理解すれば、最も重要な情報を視線が最初に到達する場所に正確に配置できます。

### Z型パターン

**横長（ランドスケープ）** のカバーに最適です。視線は左上から始まり、右上へ移動し、対角線で左下へ切り下がり、右下へ向かう――Zを描きます。

| 位置 | 配置する要素 |
|----------|-------------------|
| 左上 | ブランドマークまたは注目を引く要素 |
| 右上 | 見出しまたは主要キーワード |
| 左下 | 補足情報またはフックテキスト |
| 右下 | CTA（行動喚起）またはロゴ |

### F型パターン

**縦長（ポートレート）** のカバーに最適です。小紅書の最も一般的なカバー比率は3:4です。視線は上部を横に読み、下方向にスキャンし、中央左あたりで再び横に読む――F字を形成します。

!!! info "コンセプト"
    縦長カバーでは、**上部の2行が最も価値の高いエリア**です。見出しをここに配置して最大のインパクトを狙いましょう。中央部分はメイン画像や商品に最適で、下部は補足情報に適しています。

### 流れのデザインの基本原則

1. **自然な読む習慣に従う** ――視線をあちこちジャンプさせない
2. **重要な情報は流れの起点に配置** （左上または上部中央）
3. **補足情報は流れの終点に配置** （右下または下部）

## 情報密度

情報密度とは、1枚のカバーがどれだけの内容を伝えようとしているかです。初心者が最も間違えやすい領域の一つです。

!!! warning "注意"
    「この商品は色もきれいで、テクスチャーも滑らかで、持ちもよくて、しかも安くて、さらにサンプルも付いて…」すべてのセールスポイントをカバーに載せたくなりますが、ユーザーはどれも覚えてくれません。

**黄金ルール：1枚のカバーには1つのメッセージ。**

自分が小紅書を閲覧するときを思い出してください。親指でフィードを飛ばすとき、1枚のカバーに与える注目は1秒未満です。その短い時間で、いくつのアイデアを吸収できるでしょう？答えは1つです。

!!! tip "ポイント"
    カバーのメッセージを一文で要約してみましょう。その文が10文字以上になるなら、詰め込みすぎです――削りましょう。

| 密度 | 効果 | 適した用途 |
|---------|--------|----------|
| 低（メッセージ1つ） | 視覚的インパクト大、高いクリック率 | 商品紹介、コーデ共有、単品レビュー |
| 中（2〜3つのポイント） | バランス型、ただしレイアウト力が必要 | チュートリアル、ガイド、比較レビュー |
| 高（4つ以上のポイント） | 散らかりやすい、上級者のデザイン力が必要 | まとめ記事、リスト型投稿（初心者にはおすすめしない） |

## 投稿タイプ別レイアウト

コンテンツの種類によって、異なるレイアウト戦略が求められます。小紅書で最も一般的な3つの投稿タイプとそのアプローチを紹介します。

### チュートリアル

チュートリアルカバーの鍵は**ステップの明確さ**です。ユーザーは一目で「これはチュートリアルで、こういうことが学べる」と理解できるべきです。

- 上部に見出し、チュートリアルの内容を明示
- 中央部分にステップやビフォーアフターを示す2〜4枚の小画像
- 下部に「一緒にやってみよう」「保存しておこう」などの誘導文
- 番号ラベル（1、2、3）で順序を明確に

!!! example "実例"
    上部に大文字で「通勤5分メイク」。中央は2列に分割：左にすっぴん、右に仕上がり。下部に小文字で「すべてプチプラコスメ」。シンプル、明確、クリックしたくなるカバーです。

### 商品紹介

商品紹介カバーの鍵は**商品を主役にすること**。他のすべては脇役です。

- 商品がフレームの60%以上を占めるように
- 背景はクリーンに――単色またはシンプルなセッティング
- テキストは最小限に：ブランド名＋一行のフック
- クローズアップやディテール写真で購買意欲を刺激

### ガイド・まとめ

ガイドカバーは本質的に情報量が多いため、鍵は**見える構造**です。ユーザーに「情報は多いが、よく整理されている」と感じさせましょう。

- 番号付きリストで秩序感を演出
- 仕切り線や色ブロックでセクションを分離
- 見出しでカバーの範囲を明示――例：「東京5日間完全ガイド」
- アイコン（交通、宿泊、グルメ）でカテゴリを一目で伝達

!!! tip "ポイント"
    詳細な旅程は投稿本文に任せましょう。カバーに必要なのは「このガイドは何をカバーしているか」と「タップする価値がある」の2点だけです。ガイドの全内容を1枚の画像に詰め込もうとしないこと。

## まとめ

情報階層とレイアウトは、カバーデザインパズルの最後のピースです。以下の基本原則を大切に：

1. **明確な階層を構築** ――見出し > 小見出し > 本文 > 補足情報
2. **視覚的な流れに従う** ――最も重要な情報を視線が最初に到達する場所に配置
3. **密度をコントロール** ――1枚のカバーに1つのメッセージ
4. **コンテンツタイプに合わせる** ――チュートリアルには明確さを、商品紹介にはヒーローショットを、ガイドには構造を

3つのチャプターを合わせれば、完全なカバーデザインの方法論が手に入ります。あとは実践、観察、そして改善の繰り返しです。

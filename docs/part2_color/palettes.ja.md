# 2.2 スタイル別パレット

理論はここまで。ここからは最も実践的なパートです。スタイル別に整理されたパレットをヘックスコード付きでお届けします。カバー、背景、テキストにすぐに使えます。あなたのコンテンツの雰囲気に合うグループを見つけて、今日から使い始めましょう。

## モランディトーン

モランディカラーの本質：**すべての色が、柔らかなグレーのフィルターを通したように見えること**。低彩度、適度な明度。穏やかで洗練されていて、決して派手ではない。小紅書における「上品さ」の代名詞です。

## 高彩度コントラスト

モランディの対極――高彩度コントラストは**大胆なビジュアルインパクト**がすべてです。2つの鮮やかな対照色がぶつかり合い、瞬時に注目を集めます。画面から飛び出すようなコンテンツに最適です。

!!! warning "注意"
    コントラスト＝「2つの派手な色でキャンバスを50/50に分ける」ではありません。一方を主色（60-70%）、もう一方をアクセント（30-40%）として使いましょう。この階層がないと、鮮烈ではなく混沌になります。

**おすすめ用途：** メイクチュートリアル、グルメレビュー、ストリートウェア、イベントプロモーション

**配色のコツ：**

- 一方の色でキャンバスの大部分を占め、もう一方をタイトルや局所的なアクセントに使用
- 白やライトグレーを緩衝ゾーンとして挟み、視覚的な緊張を和らげる
- 高コントラストパレットは画像カバーの方がテキスト多めのレイアウトより効果的

## クリームトーン

キーワードは：**柔らかい、温かい、ふわふわ、心地よい**。すべての色がクリーム色のフィルターを通したように見えて、深く癒される。韓国系・日本系クリエイターの定番パレットです。

    クリームトーンでカバーを作るとき、テキストは**ダークブラウン**（例：#5C4033）や**ダークスレート**（例：#4A5568）を使いましょう。純粋な黒はコントラストが強すぎて、柔らかさが壊れてしまいます。

**おすすめ用途：** 日系コーデ、韓国カフェレビュー、癒し系Vlog、手帳シェア、スイーツ作り

**配色のコツ：**

- 全体を高明度・低彩度の範囲に収める
- クリームトーン同士は自由に組み合わせても衝突しない
- テキストにはダークブラウンやグレー系を使用。純黒は避ける
- 特に縦長カバーと統一されたグリッド美学に適している

## ダーク＆ゴールド

ダーク＆ゴールドが伝えるのは：**品質、力強さ、プロフェッショナリズム**。深い背景にゴールドのアクセントが、自然とプレミアム感を醸し出します。テック、ビジネス、メンズ系コンテンツの定番パレットです。

!!! info "コンセプト"
    黒はミステリーと力を象徴し、ゴールドは富と品質を表します。この2色の組み合わせは強い明暗のコントラストを生み、ゴールドの持つ温かみが黒の冷たさを和らげます。結果として、力強さと親しみやすさを兼ね備えた印象に。

**おすすめ用途：** テックガジェット、ビジネスインサイト、メンズファッション、高級品開封レビュー、金融系コンテンツ

**配色のコツ：**

- 背景には#1A1A1Aを使用。純粋な#000000よりも質感があります
- ゴールドはタイトル、アイコン、区切り線などの小面積に限定
- チャコールをトランジションカラーとして使い、レイアウトが二色だけに見えないように
- 本文テキストは白またはライトグレーで可読性を確保

## クイックリファレンス

| パレット | キーワード | 彩度 | 適したコンテンツ |
|---------|----------|-----------|-------------|
| モランディ | 洗練、くすみ、穏やか | 低 | インテリア、コーデ、ライフスタイル |
| 高彩度コントラスト | 大胆、鮮やか、目を引く | 高 | ビューティー、グルメ、ストリートウェア |
| クリーム | 柔らかい、癒し、温かい | 低〜中 | 日系、韓国系、スイーツ |
| ダーク＆ゴールド | 高級感、力強さ、プロフェッショナル | 低〜中 | テック、ビジネス、メンズ系 |
